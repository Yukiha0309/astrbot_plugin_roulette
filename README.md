# AstrBot 简单的轮盘赌

群聊多人轮盘赌插件，默认支持 2 到 6人，也可通过配置调高人数上限；包含弹仓轮、公开道具和房主处决挂机玩家。

## 基础规则

- 2 人局：5 血；第 3 个弹仓轮开始每人发 2 个道具；背包上限 6；前 2 个弹仓轮最多 4 发。
- 3/4 人局：3 血；第 3 个弹仓轮开始每人发 2 个道具；背包上限 4；前 3 个弹仓轮最多 6 发。
- 5 人及以上：2 血；第 2 个弹仓轮开始每人发 1 个道具；背包上限 3；开局即最多 10 发。
- 弹仓每轮 3 到 10 发，至少 1 发实弹、1 发空弹；公开数量，不公开顺序。
- 道具只能在自己的行动回合使用。被跳过回合时不能使用道具、不能开枪。
- 放大镜结果公开到群聊。
- 房主或超级管理员可以处决挂机玩家，插件不做自动倒计时。
- 对玩家造成伤害后会提示剩余生命。
- 轮到玩家行动时会 @ 提醒。
- `/轮盘状态` 不再显示弹仓剩余信息，以保留危机感。
- `/查看道具` 可查看自己的道具。
- `/死亡榜` 会统计本群玩家死亡次数。
- `/梭哈` 只显示确认提示，不直接执行。
- `/梭哈 数量` 可以指定最多使用几发，遇到实弹会提前停止。
- `/梭哈 all` 会使用当前弹仓剩余全部子弹。
- 梭哈执行过程不会跨弹仓；如果全部为空弹并打空当前弹仓，会进入下一弹仓轮且仍由当前玩家继续行动。
- 梭哈时加压弹簧只影响第一枪：第一枪空弹会炸膛扣血但继续梭哈，第一枪实弹则按加压伤害结算并切换回合。
- 默认使用玩家代号 P1、P2 等，避免群名过长影响阅读。

## 异界战争人数规则

| 人数 | 血量 | 道具开始 | 每次发放 | 背包上限 | 前期弹仓限制 |
| --- | ---: | ---: | --- | ---: | --- |
| 2 人 | 5 | 第 3 轮 | 每人随机 2-4 个 | 8 | 前 3 轮最多 4 发 |
| 3-4 人 | 4 | 第 2 轮 | 每人随机 2-3 个 | 6 | 前 2 轮最多 5 发 |
| 5-6 人 | 3 | 第 1 轮 | 每人随机 1-2 个 | 4 | 前 2 轮最多 7 发 |
| 7 人及以上 | 2 | 第 1 轮 | 每人 1 个 | 3 | 开局即最多 10 发 |

## 道具

| 道具 | 效果 |
| --- | --- |
| 魔镜 | 公开查看当前子弹是真弹还是空弹 |
| 止痛片 | 回复 1 血，不超过血量上限 |
| 肘击 | 被牢大肘击后，退掉当前子弹并公开类型 |
| 束线带 | 指定一名存活玩家跳过下一次行动，不能叠加 |
| 改装工具 | 下一枪若为实弹则伤害 +1 |
| 反转器 | 反转当前子弹，实弹变空弹，空弹变实弹 |
| 测弹仪 | 随机显示弹仓中某一位置的子弹类型 |
| 顺手牵羊 | 随机偷取目标 1 个普通道具并立即使用 |
| 怪味蘑菇 | 50% 回复 2 血，50% 受到 1 点伤害 |

## 异界战争道具

| 道具 | 效果 |
| --- | --- |
| 凶吉签 | 查看当前子弹，凶为实弹，吉为空弹 |
| 绷带 | 回复 1 血并解除自身流血 |
| 酒狐委托 | 酒狐往弹仓随机位置加入 1 发未知子弹，可能是真也可能是假 |
| 封膛符 | 目标下次行动可以使用道具，但不能真正开枪 |
| 锈蚀枪管 | 下一次实弹命中玩家时附加流血 |
| 加压弹簧 | 下一枪实弹伤害 +1，空弹则自己炸膛受 1 点伤害 |
| 隙间之手 | 随机偷取目标 1 个普通道具 |

## 配置项

| 配置项 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `super_admins` | list | `[]` | 超级管理员 QQ 号，可结束游戏或处决挂机玩家 |
| `whitelist_groups` | list | `[]` | 群聊白名单；填写后仅这些群可用 |
| `blacklist_groups` | list | `[]` | 群聊黑名单 |
| `max_players` | int | `6` | 最大玩家数。默认 6，可调高到 7 人或以上，但超过 6 人后平衡性与回合节奏可能下降 |
| `use_player_codes` | bool | `true` | 是否使用 `P1`、`P2` 等玩家代号 |
| `player_name_max_length` | int | `8` | 玩家昵称截断长度 |
| `jiuhu_event_enabled` | bool | `true` | 是否启用酒狐事件，仅异界战争模式生效 |
| `jiuhu_event_chance` | int | `30` | 每个弹仓轮开始时触发酒狐事件的概率，仅异界战争模式生效 |
