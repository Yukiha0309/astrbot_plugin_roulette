# AstrBot 简单的轮盘赌

群聊多人轮盘赌插件，支持 2 到 6 人、弹仓轮、公开道具和房主处理超时玩家。

## 指令

| 指令 | 说明 |
| --- | --- |
| `/轮盘创建` | 在当前群创建房间 |
| `/轮盘加入` | 加入等待中的房间 |
| `/轮盘开始` | 房主或超级管理员开始游戏 |
| `/开自己` | 对自己开枪 |
| `/开 @用户` | 对指定存活玩家开枪 |
| `/使用道具 道具名` | 使用道具 |
| `/使用手铐 @用户` | 道具短指令，其他道具同理 |
| `/梭哈` | 查看梭哈确认提示，不直接执行 |
| `/梭哈 数量` | 最多使用指定数量，不跨弹仓 |
| `/梭哈 all` | 梭哈当前弹仓剩余全部子弹 |
| `/查看道具` | 查看自己的道具 |
| `/轮盘状态` | 查看当前房间状态 |
| `/死亡榜` | 查看本群死亡次数排行 |
| `/轮盘处决 @用户` | 房主或超级管理员处决挂机玩家 |
| `/轮盘结束` | 房主或超级管理员结束当前房间 |
| `/轮盘帮助` | 查看帮助 |

## 游戏规则

- 2 人局：5 血；第 3 个弹仓轮开始每人发 2 个道具；背包上限 6；前 2 个弹仓轮最多 4 发。
- 3/4 人局：3 血；第 3 个弹仓轮开始每人发 2 个道具；背包上限 4；前 3 个弹仓轮最多 6 发。
- 5/6 人局：2 血；第 2 个弹仓轮开始每人发 1 个道具；背包上限 3；开局即最多 10 发。
- 弹仓每轮 3 到 10 发，至少 1 发实弹、1 发空弹；公开数量，不公开顺序。
- 道具只能在自己的行动回合使用。被跳过回合时不能使用道具、不能开枪。
- 放大镜结果公开到群聊。
- 房主或超级管理员可以处决挂机玩家，插件不做自动倒计时。
- 对玩家造成伤害后会提示剩余生命。
- 轮到玩家行动时会 @ 提醒。
- `/轮盘状态` 不再显示弹仓剩余信息，以保留危机感。
- `/查看道具` 可查看自己的道具。
- `/死亡榜` 会统计本群玩家死亡次数。
- 修复轮到玩家时 @ 变成 CQ 文本的问题。
- `/梭哈` 只显示确认提示，不直接执行。
- `/梭哈 数量` 可以指定最多使用几发，遇到实弹会提前停止。
- `/梭哈 all` 会使用当前弹仓剩余全部子弹。
- 梭哈不会跨到下一轮弹仓。
- 默认使用玩家代号 P1、P2 等，避免群名过长影响阅读。

## 道具

| 道具 | 效果 |
| --- | --- |
| 放大镜 | 公开查看当前子弹是真弹还是空弹 |
| 香烟 | 回复 1 血，不超过血量上限 |
| 啤酒 | 退掉当前子弹并公开它是真弹还是空弹 |
| 手铐 | 指定一名存活玩家跳过下一次行动，不能叠加 |
| 短刀 | 下一枪若为实弹，伤害 +1 |


## 安装

将本仓库下载或克隆到 AstrBot 的插件目录中：

```bash
git clone https://github.com/Yukiha0309/astrbot-plugin-roulette.git
```
然后重启 AstrBot，或在 AstrBot 插件管理页面重新加载插件。

## 配置项

- `use_player_codes`：是否使用 `P1`、`P2` 等玩家代号，默认开启。
- `player_name_max_length`：玩家昵称截断长度。
